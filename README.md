# 🍀POOMY
소품(poom) + Lucky : 소품샵 찾아주는 앱, 푸미입니다. 

장소, 분위기 기반 소품샵 필터를 통해 사용자의 취향을 반영한 맞춤 소품샵을 추천해줍니다.  
소품샵 리뷰와 에디터의 뉴스레터를 통해 생생한 소품샵 후기들을 보실 수 있습니다. 

푸미(Poomy)와 함께 일상에 특별함을 더해보세요!

## 🍀상세설명

1. 로그인 & 사용자 등록 페이지
2. 메인 페이지 & 소품샵 검색 페이지
3. 소품샵 상세 페이지
4. 마이페이지 & 소품샵 찜 페이지
5. 유저 리뷰 작성 페이지
6. 장소/키워드 추천 페이지 & 뉴스레터 페이지
